# Intersection Observer Demo

Next.js(TypeScript)でIntersection Observer APIの使用方法を学ぶためのデモアプリケーションです。

## 機能

このデモでは、Intersection Observer APIを使用して以下の機能を実装しています：

- ビューポートに入った要素の検出
- 要素の交差率（表示されている割合）のリアルタイム表示
- 交差率に応じた動的な色の変化

## 技術スタック

- [Next.js](https://nextjs.org/) - Reactフレームワーク
- [TypeScript](https://www.typescriptlang.org/) - 型安全なJavaScript
- [Tailwind CSS](https://tailwindcss.com/) - ユーティリティファーストCSSフレームワーク

## 開発方法

```bash
# 依存関係のインストール
npm install

# 開発サーバーの起動
npm run dev

# 本番ビルド
npm run build

# 本番サーバーの起動
npm start

# リントの実行
npm run lint
```
